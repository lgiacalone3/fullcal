A module for CiviCRM that displays events in an interactive calendar in Joomla, using FullCalendar by Adam Shaw (arshaw.com) and extending the code base of CiviEvent Joomla Module. 

The CiviEvent Joomla Module (http://wiki.civicrm.org/confluence/display/CRMDOC42/CiviEvent+Joomla+Module) was originally developed by Nakul Ganesh as part of the 2009 Google Summer of Code Program and further developed by Brian Shaughnessy before being leveraged here.

To download an edge release, use the following link: https://github.com/lgiacalone3/fullcal/archive/master.zip
